# RISCV2617-main

The design is developed in a separate repository using a different directory structure.
The repository can be found here: https://codeberg.org/zeri42/prjleisurecruise

The files included here are copied directly from the source repo without adjustments to make them work with this directory structure.
Also the top-level name differs, since we do not (want to) use the generated IHP name for our development.
Still, the files given here contain all information needed to replicate the tapeout.

The RTL is generated from SpinalHDL, the SpinalHDL sources can be found in the referenced source repository.
The generated RTL as used in the tapeout is included here.
The RTL-level tests can be found in and simulated from the SpinalHDL sources as well.
